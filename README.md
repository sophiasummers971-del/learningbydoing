## Installation

<table>
<tr>
<td>

### One-liner (recommended)

```bash
curl -sSL https://raw.githubusercontent.com/Z4nzu/hackingtool/master/install.sh | sudo bash
```

Handles everything — prerequisites, clone, venv, launcher.

</td>
<td>

### Manual Install

```bash
# Clone the repository
git clone https://github.com/Z4nzu/hackingtool.git
cd hackingtool

# Run the installation script with sudo to set up everything correctly
sudo python3 install.py
```

This script detects your local source and sets up the necessary environment.

Then run: `hackingtool`

</td>
</tr>
